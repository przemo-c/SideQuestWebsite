Guide to Downloads

For Windows 8 and above, it is recommended to download the Setup EXE file. Alternately, you may download the ZIP file, which contains the same contents without an installer. For 32bit there is a zip marked IA32-WIN.ZIP. Windows 7 is not fully supported.

For users of OS X / macOS 10.12+, the DMG file is the recommended option. Alternately, you may download the ZIP file. For both files, once downloaded, double-click and move the SideQuest application to your Applications folder.

Linux users, you can download the TAR.XZ file. You may need to install additional packages from your OS. The names of these packages will vary depending on the distribution you are using.
