SideQuest Scripting in C-Sharp

Back To Developers
Get going with scripting in SideQuest to open a whole world of possibilities to create content.

Notice: This next section assumes you are familiar with Unity3D and C-Sharp scripting. Unity has a lot of great tutorials to get your started.

You can build a whole range of cool things with scripting. At SideQuest Legends we thought it was important to build a scripting SDK that was as easy as possible to use but also had to be powerful and flexible. We decided early on that the most beneficial option was to allow users to use full C-Sharp scripting like you already do in Unity3D. We have created a scripting enabled SDK that allows you to run your scripts on our platform with a host of benefits.
What you can do
doneAccess most of the Unity3D API with a few extras like Json.NET and websocket-sharp.
doneLeverage our integrated interaction system for super simple user interactions like grabbales, clickables and spawners.
doneInteract with the API as permitted by the user.
doneLeverage networking to sync events / data between users.


What you can't do
closeUse DLLs with your scripting or access any other assemblies other than those provided by us.
closeAccess secure namespaces like IO, Reflection, Process, Input.
