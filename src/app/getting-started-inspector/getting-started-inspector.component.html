Getting Started - Play test with the SideQuest Inspector

Run your SideQuest Legends App inside Unity3D.



Step 1 - Select Squanch and Enter Space ID
Open the SideQuest option in the Tools menu of Unity3D. Enter a Space ID to launch Legends in a specific space - this is good for testing against other apps created with Legends. Select a squanch file from your hard drive and load it into the selected space. The unity editor will automatically play once you select a file.


Step 2 - Re-running the same Squanch
Once you have selected a squanch file it will be added to the list of recent squanches selected - this makes it easier to load the same squanch later.
